## Program

```typescript
#include <stdio.h>

int sod(int n) {
  if (n == 0) return 0;
  return n % 10 + sod(n / 10);
}

int main() {
  int digit, num;
  printf("Enter the number: ");
  scanf("%d", &num);
  printf("Sum of digits of %d is %d", num, sod(num));
}
```
