## Program

```c
#include <stdio.h>

int main() {
  int num, i, j;
  printf("Enter the size: ");
  scanf("%d", &num);

  for (i = 1; i <= num; i++) {
    for (j = 1; j <= i; j++) {
      printf("%d ", i);
    } 
    printf("\n");
  }
}
```
