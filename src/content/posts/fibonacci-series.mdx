## Program

```c
#include <stdio.h>

int main() {
  int first = 0, second = 1, next, n;

  printf("Enter number of elements needed: ");
  scanf("%d", &n);

  for (int i = 0; i < n; i++) {
    if (i <= 1) {
      printf("%d ", i);
    } else {
      next = first + second;
      first = second;
      second = next;
      printf("%d ", next);
    }
  }
}
```
