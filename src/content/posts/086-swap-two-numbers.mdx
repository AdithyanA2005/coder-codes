## Program

```bash
#!/bin/bash

read -p "Enter first number: " num1

read -p "Enter second number: " num2

echo "Before swapping: num1 = $num1, num2 = $num2"
temp=$num1
num1=$num2
num2=$temp
echo "After swapping: num1 = $num1, num2 = $num2"
```
